# 🗑️ Garbage Collection Tracker

A smart web-based system to track garbage pickups and send timely reminders.

---

## 🚀 Features

- 📅 Add pickup schedules
- 🔔 Smart notifications (before arrival & at pickup time)
- ❌ Track missed pickups automatically
- 📊 Daily status tracking

---

## 🛠️ Tech Stack

- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript
- Storage: JSON (file-based)

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
